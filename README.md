## О себе
Аналитик с опытом в управлении запасами, прогнозировании спроса и BI. Строил ETL-пайплайны с нуля, работал с SQL, Power BI, DataLens.

## Проекты

### 1. ETL-пайплайн для финансовой отчётности маркетплейсов
**Задача:** Автоматизировать сбор и обработку данных по продажам, остаткам и финансам с двух маркетплейсов (WB, Ozon).

**Решение:**
- Спроектировал архитектуру пайплайна: API → DWH → SQL → Excel
- Настроил автовыгрузку через планировщик Windows
- Реализовал расчёт себестоимости, комиссий, логистики и маржинальности

**Результат:** Отчёт, который раньше делали 2 дня вручную, обновляется за 15 минут.

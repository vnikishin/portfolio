## Проекты

### 1. ETL-пайплайн для финансовой отчётности маркетплейсов
**Задача:**  
Автоматизировать расчёт себестоимости, маржинальности и логистики для трёх маркетплейсов (WB, Ozon, Яндекс) — от выгрузки до готового отчёта.

**Что сделал:**
- Спроектировал архитектуру ETL-пайплайна:
  - `self_cost` (Excel) → загрузка в DWH
  - `options2provide_from_1C` (1С) → загрузка в DWH
  - SQL-скрипты → расчёт себестоимости, комиссий, логистики, маржинальности
  - Автоматический расчёт итоговых отчётов для руководства
- Настроил оркестрацию через планировщик Windows
  
📁 [Схема ETL-пайплайна](etl_pipeline.drawio.pdf.pdf)
